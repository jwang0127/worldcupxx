# 世界杯预测复盘与出稿逻辑

这份文档给接手的人直接用，目标不是解释技术，而是保证别人拿到后能按同一套流程继续稳定输出:

1. 先复盘昨日赛果
2. 再做今日 4 场预测
3. 生成可分享的中文页面或文案
4. 出稿前完成一轮强制检查

## 一句话说明项目在做什么

这是一个“每日世界杯预测看板”项目。

每天根据:

1. 官方或已确认的赛程
2. 当天体彩/赔率材料
3. 昨日真实赛果

输出一份中文预测复盘内容，核心包括:

1. 昨日命中情况
2. 今日每场总进球预测
3. 每场两个稳胆比分和一个冷门比分
4. 五段式分析
5. 组合建议
6. 风险提示

## 现在的硬规则

### 1. 赛程表是最高优先级基准

以后所有预测前，必须先以 `2026年世界杯赛程表.xlsx` 为准。

要先核对:

1. 场次对应的主队
2. 场次对应的客队
3. 北京时间
4. 当地时间
5. 所属小组

赔率图只能用来补赔率，不能代替赛程核对。

### 2. 体彩场次号不能直接等同于赛程顺序号

这次已经证明，体彩当天场次号、截图里的展示顺序、总赛程表顺序，可能并不完全一一对应。

所以不能再用下面这种危险做法:

1. 看到一个场次号
2. 凭记忆补球队
3. 不核对赛程表就直接出稿

### 3. 玄学起卦必须用比赛当地时间

以后玄学部分的起卦、流时、时辰判断，统一使用比赛当地时间，不再使用北京时间直接起卦。

页面里可以同时展示:

1. 北京时间
2. 当地时间

但玄学判断以当地时间为准。

## 这次 035 出错的真实原因

### 错误现象

原稿把 `035` 写成了:

`厄瓜多尔 vs 克罗地亚`

核对赛程表后应为:

`厄瓜多尔 vs 库拉索`

### 根因

这次不是单纯手误，而是流程错误，具体有 3 个原因:

1. 出稿前没有先用 `2026年世界杯赛程表.xlsx` 强制校对主客队
2. 把体彩场次号默认当成赛程顺序号处理，产生了串位风险
3. 在赔率材料里能看到盘口强弱，但不能可靠推导客队名称，仍然做了人工补写

### 修正结论

以后必须先过赛程表，再写赔率分析。

顺序固定为:

1. 先确认赛程
2. 再录入赔率
3. 再生成分析
4. 最后出稿前复查一次

## 每天固定执行顺序

### 第 1 步: 先确认日期和今日场次

先确认今天要处理的是哪一天，以及今日 4 场比赛编号。

### 第 2 步: 先查赛程表

打开 `2026年世界杯赛程表.xlsx`，对这 4 场逐一确认:

1. 主队
2. 客队
3. 北京时间
4. 当地时间
5. 小组

如果赛程表与手头图片、文字、旧数据不一致:

以赛程表为准，再去修正数据文件。

### 第 3 步: 再录入赔率

优先从体彩截图或赔率材料录入:

1. 胜平负
2. 让球胜平负
3. 比分
4. 总进球
5. 半全场

### 第 4 步: 回填昨日真实赛果

复盘至少看两件事:

1. 总进球是否命中
2. 胜平负方向是否命中

### 第 5 步: 根据昨日复盘修正今日逻辑

现在项目的核心修正规则:

1. 强队盘不能机械照抄最低总进球赔率
2. 强队领先后的 3-4 球尾部要提高权重
3. 弱队偷一球、补时波动、下半场提速要纳入
4. 均势盘要保留 `1:1`、`0:1` 这类冷门防线
5. 胜平负方向可以参考最低赔，但不能默认深穿

### 第 6 步: 生成今日 4 场预测

每场固定输出:

1. 胜平负方向
2. 总进球主线
3. 两个稳胆比分
4. 一个冷门比分
5. 置信度

### 第 7 步: 写五段式详细分析

每场都必须有:

1. 基本面
2. 战术与教练
3. 外部因素
4. 小组形势
5. 赔率解读

### 第 8 步: 写玄学分析

玄学固定嵌入每场卡片里，不单独拆页。

可以覆盖:

1. 周易梅花 / 六爻
2. 紫微流时
3. 奇门遁甲
4. 干支五行 / 飞星
5. 诸葛神数 / 塔罗

注意:

1. 玄学仅作为娱乐表达
2. 起卦时辰必须按当地时间
3. 页面底部必须保留理性提示

### 第 9 步: 给出组合建议

默认给两组:

1. 三串一总进球
2. 三串一比分

## 出稿前强制检查清单

以后每次输出前，必须逐条检查，不能跳过:

1. `data/YYYYMMDD.json` 里的主客队是否与 `2026年世界杯赛程表.xlsx` 一致
2. 每场的北京时间是否与赛程表一致
3. 每场的当地时间是否已写入并可用于玄学判断
4. 赔率是否填到了正确的场次，不存在串场
5. 页面里的主客队、时间、组别是否和 JSON 一致
6. `review.html`、`index.html`、`predict_YYYYMMDD.html` 三处是否同步
7. 玄学部分是否使用当地时间而不是北京时间
8. 是否再次检查过一次易错场次，尤其是人工补录的比赛

## 对外分享时建议的固定结构

如果是发群、发朋友圈、发公众号，建议按下面顺序写:

### 今日四场结论

每场一行:

`033 荷兰 vs 瑞典：3球，2:1 / 3:0，防 1:1`

### 今日稳胆

写最有把握的一场或一个总进球主线。

### 今日最大冷门

写最需要防的平局或客偷比分。

### 昨日复盘

写清:

1. 哪几场命中
2. 哪几场偏差
3. 今天做了什么修正

### 风险提示

固定保留:

`以上仅为公开信息整理后的娱乐分析，不构成任何购彩建议，请理性参考。`

## 给接手人的一句话

这套逻辑最重要的不是“猜比分”，而是先把赛程核准，再用赔率定主线，用复盘修正偏差，用冷门防线防止机械跟热。

以后遇到任何人工补录场次，先看赛程表，再看赔率图，最后才允许出稿。
