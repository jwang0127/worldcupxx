# 世界杯预测复盘逻辑分享版

这是一份可以直接转发给别人使用的说明文档。

目标不是讲技术细节，而是让接手的人拿到之后，能够按同样逻辑持续产出：

1. 昨日赛果复盘
2. 今日四场比赛预测
3. 可分享的中文页面或中文文案
4. 能继续迭代的“复盘修正”逻辑

---

## 一句话说明项目在做什么

这是一个“每日世界杯预测看板”项目。

每天根据：

1. 当天比赛赛程
2. 体彩或截图里的赔率
3. 昨日真实赛果

输出一份中文预测复盘内容，核心包括：

1. 昨日预测命中情况
2. 今日每场比赛的总进球预测
3. 每场两个稳胆比分和一个冷门比分
4. 五段式分析
5. 组合推荐
6. 风险提示

---

## 最终要产出的内容长什么样

每天的成品必须是“给普通人一眼能看懂”的中文内容，而不是流程图、模型图或者抽象术语堆砌。

最终内容固定分成这几块：

1. 昨日复盘与今日模型修正
2. 今日预测结果总表
3. 今天该买什么
4. 每日总结页
5. 每场详细分析
6. 三串一总进球推荐
7. 三串一比分推荐
8. 风险提示与备选

---

## 每天的工作顺序

每天按下面顺序执行，不要乱。

### 1. 先确认日期和当天场次

先确定今天是哪一天，以及今天要预测哪 4 场比赛。

例子：

- `2026-06-20`
- 今日场次：`029`、`030`、`031`、`032`

### 2. 收集今天的赔率

优先使用体彩赔率截图，至少要拿到这几类信息：

1. 胜平负
2. 让球胜平负
3. 比分
4. 总进球
5. 半全场

如果不是截图，而是手工录入，也至少要保证：

1. 主胜赔率
2. 平局赔率
3. 客胜赔率
4. 总进球 `0-7+`
5. 若有，让球盘一并记录

### 3. 回填昨日真实赛果

昨日四场比赛先回填真实比分。

复盘至少看两件事：

1. 总进球是否命中
2. 胜平负方向是否命中

### 4. 根据昨日复盘修正今日逻辑

这一步最重要。

不是每天机械照抄最低赔率，而是要根据昨天的偏差修正今天。

当前项目已经形成的核心修正规则是：

1. 如果总进球大量偏离，不能只盯最低总进球赔率
2. 强队盘要提高 3-4 球尾部的重视
3. 弱队偷一球、补时波动、下半场提速要纳入考虑
4. 均势盘要降低置信度，保留 `1:1`、`0:1` 这类冷门防线
5. 胜平负方向可以继续参考最低赔，但不要默认深穿

### 5. 生成今日四场预测

每场比赛固定输出：

1. 胜平负方向
2. 总进球主线
3. 两个稳胆比分
4. 一个冷门比分
5. 置信度

### 6. 写详细分析

每场必须有五段分析，不能只给结论。

固定五段如下：

1. 基本面
2. 战术与教练
3. 外部因素
4. 小组形势
5. 赔率解读

### 7. 把玄学分析嵌进单场卡片

玄学分析不能独立成一页，必须嵌在每场比赛卡片里。

可用角度包括：

1. 周易梅花 / 六爻
2. 紫微流时
3. 奇门遁甲
4. 干支五行 / 飞星
5. 诸葛神数 / 塔罗

注意：

1. 玄学只作为娱乐分析表达
2. 最终页面必须保留理性购彩提示

### 8. 组合推荐

每天默认给两组：

1. 三串一总进球
2. 三串一比分

组合逻辑：

1. 总进球组合优先选赔率与模型同向的 3 场
2. 比分组合优先选每场第一稳胆比分
3. 如果是强客盘，可给更高的比分估算赔率

## 每场比赛的判断逻辑

下面是别人最需要直接照着用的部分。

### 1. 胜平负方向怎么定

先看胜平负最低赔落在哪一侧：

1. 主胜最低赔，先偏主胜
2. 客胜最低赔，先偏客胜
3. 平局赔率明显压低，不能把比赛写成单边

然后再看让球盘修正：

1. 如果让一球后，强方没有明显继续占优，说明“赢球可以，深穿未必”
2. 如果让球后弱方赔率更低，说明净胜空间有限
3. 如果让两球主胜仍很低，属于重压制盘

### 2. 总进球怎么定

先看总进球赔率最低点。

但不能只机械选最低点，要结合昨日复盘做修正：

1. 强队盘最低点在 `2球` 或 `3球` 附近时，要警惕上修到 `3球` 或 `4球`
2. 均势盘最低点在 `2球`，通常保留 `2球`
3. 若平局赔率低、比分盘 `1:1` 最低，通常优先保留 `2球`
4. 若让球深、比分盘 `3:0`、`4:0` 都低，可以抬高到 `4球`

### 3. 稳胆比分怎么定

优先从比分赔率低位里挑两个，同时要和胜平负方向、总进球主线一致。

例子：

1. 主强小胜盘：`2:0 / 2:1`
2. 客强窄胜盘：`0:1 / 0:2`
3. 强压制盘：`3:0 / 4:0`
4. 均势盘：`1:1 / 2:1`

### 4. 冷门比分怎么定

冷门比分不是乱写，通常从这几个脚本里选：

1. `1:1`
2. `0:1`
3. `2:1` 或 `1:2`
4. 强队被偷一球的 `2:1`

原则：

1. 强队盘常防平
2. 均势盘常防客偷
3. 高热主队盘防“赢球不穿”或“闷平”

## 五段式详细分析怎么写

下面给一个固定写法，别人直接套就能用。

### 1. 基本面

写法重点：

1. 先讲赔率给出的强弱排序
2. 再讲这是不是绝对碾压盘
3. 最后落到总进球和比分主线

示例句式：

`主胜 1.45 明显占优，但让一球没有完全站向主队，说明市场更认可赢球而非大胜。结合比分盘低位，本场更像 2 球主线的小胜结构。`

### 2. 战术与教练

写法重点：

1. 谁会主动控球
2. 谁适合反击
3. 哪种局面会把比赛推向第二个稳胆比分

示例句式：

`主队更可能先控球压迫，但如果迟迟打不开，对手的转换反击会把脚本推向 2:1 或 1:1。`

### 3. 外部因素

写法重点：

1. 开球时间
2. 中立场
3. 时差、体能、天气、节奏

示例句式：

`凌晨场更容易先谨慎后提速，中立场不额外给主队加成，因此外部因素更支持小胜，而不支持过热的大比分打穿。`

### 4. 小组形势

写法重点：

1. 是不是六分战
2. 是否有净胜球需求
3. 平局是否可接受

示例句式：

`这类比赛对强队不仅是拿三分，更是争净胜球；但对弱队来说，若能拖住前 60 分钟，1 分价值极高。`

### 5. 赔率解读

写法重点：

1. 直接把胜平负和总进球低位读出来
2. 点出盘口真正想表达的内容
3. 给出最终结论

示例句式：

`盘口核心不是主胜能不能出，而是能不能深穿。总进球低位集中在 2 球和 3 球之间，因此模型不追大开，先看 2 球主线。`

## 复盘部分怎么写

复盘不是只写“中了几场”，而是要把第二天的修正说清楚。

复盘固定分三层：

### 1. 结果表

每场写：

1. 预测总进球
2. 最终比分
3. 总进球命中/未中
4. 方向命中/偏离

### 2. 汇总数据

至少要有：

1. 已结算几场
2. 总进球命中几场
3. 胜平负方向命中几场
4. 命中率

### 3. 模型反馈

模型反馈必须能指导第二天。

可直接用这套表达框架：

1. 如果总进球差：
   `说明不能只盯最低总进球赔率，需要提高尾部进球和补时波动权重。`
2. 如果胜平负差：
   `说明不能机械站边，需要提高平局和一球冷门的保护。`
3. 如果强队频繁打大：
   `说明强队领先后未必收死，3-4 球路径要上调。`

## 对外分享时最建议保留的固定格式

如果别人不是直接生成网页，而是想发群、发朋友圈、发公众号，建议固定写成下面这种顺序：

### 今日四场结论

1. `029 美国 vs 澳大利亚：2球，2:0 / 2:1，防 1:1`
2. `030 苏格兰 vs 摩洛哥：2球，0:1 / 0:2，防 1:1`
3. `031 巴西 vs 海地：4球，3:0 / 4:0，防 2:1`
4. `032 土耳其 vs 巴拉圭：2球，1:1 / 2:1，防 0:1`

### 今日稳胆

写最有把握的一场或一个总进球。

### 今日最大冷门

写最需要防的平局或客偷比分。

### 昨日复盘

写：

1. 哪场命中
2. 哪场偏差
3. 今天做了什么修正

### 风险提示

固定保留一句：

`以上仅为公开信息整理后的娱乐分析，不构成任何购彩建议，请理性参考。`

## 最后给接手人的一句话

这套逻辑的核心不是“猜比分”，而是：

1. 用赔率确定主线
2. 用复盘修正偏差
3. 用详细中文分析把结论讲清楚
4. 用冷门防线避免机械跟热

如果每天都按这个顺序做，内容会稳定很多。
